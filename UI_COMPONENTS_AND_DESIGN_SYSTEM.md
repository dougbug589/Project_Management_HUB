## What Is This?
All the buttons, forms, tables, cards you see in the app - they're all built using a consistent design system. This makes the app look professional and work smoothly.

## Tech Stack Used
- **Frontend:** React 18 + TypeScript + Tailwind CSS
- **Styling:** Utility-first CSS (no external UI library)
- **Icons:** Built-in SVG components
- **Theming:** Custom theme context system

---

## Components Available

### **Buttons**
Simple clickable elements.
- Primary button (blue) - Main actions
- Secondary button (gray) - Alternative actions
- Danger button (red) - Delete actions
- Loading state - Shows spinner while processing

### **Cards**
Container boxes for information.
- Project cards - Show project info
- Task cards - Display task details
- User cards - Profile information
- With hover effects for interactivity

### **Forms & Inputs**
For users to enter data.
- Text input - Single line text
- Text area - Multiple lines
- Dropdown - Select from options
- Checkboxes - Yes/no choices
- Date picker - Choose dates
- All inputs have validation and error messages

### **Tables**
Show data in rows and columns.
- Sortable columns - Click header to sort
- Paginated - Show multiple pages of data
- Selectable rows - Check boxes for bulk actions
- Responsive - Works on mobile too

### **Modal Dialogs**
Pop-up windows for important actions.
- Confirm deletions
- Show detailed information
- Ask for user input
- Close button to dismiss

### **Dropdown Menus**
Collapsible lists.
- Click to open/close
- Select from options
- Shows checkmark for selected item

### **Tabs**
Organize content into sections.
- Click tabs to switch views
- Keeps content organized
- Shows active tab with highlight

### **Accordion**
Expandable sections.
- Click to show/hide content
- Good for FAQs
- Saves space on page

### **Loader/Spinner**
Shows when loading data.
- Animated spinner
- Loading text message
- Used during API calls

### **Toast Notifications**
Small messages in corner.
- Success messages (green)
- Error messages (red)
- Info messages (blue)
- Auto-dismiss after 3 seconds

### **Sidebar**
Navigation menu on left.
- Navigation links
- Current page highlighted
- Collapse/expand button
- Shows user info and role

## Responsive Design
The app works on:
- Desktop (1920px and up)
- Tablets (768px - 1024px)
- Mobile (320px - 767px)

All components adjust automatically for different screen sizes.

- Screen reader friendly
- Clear contrast for readability
- Labels for all form inputs
